# FinCal Goal-Based Investment Calculator

An interactive financial planning tool that helps investors estimate the required monthly SIP to achieve future financial goals.

## Features

- Goal-based investment planning
- Inflation impact simulation
- Monthly SIP calculation
- Interactive charts
- Educational financial explanations

## Technology

- Next.js
- React
- TypeScript
- Recharts
